 Global Bearer Token — set once, used by every authenticated request

Authentication
POST /api/auth/signup Register a new user account 🔓 Public
POST /api/auth/login Login and get JWT tokens 🔓 Public

💡 After a successful login, copy the accessToken from the response and paste it in the Global Token bar above.

POST /api/auth/google Login / register via Google OAuth token 🔓 Public
GET /api/auth/me Get currently authenticated user 🔒 Auth
POST /api/auth/logout Revoke current tokens 🔒 Auth
POST /api/auth/forgot-password Request a password reset email 🔓 Public
POST /api/auth/reset-password/:token Reset password using email token 🔓 Public
POST /api/auth/change-password Change password (logged-in users only) 🔒 Auth
User Profile & Applications
GET /api/user/profile Get your profile 🔒 Auth
PATCH /api/user/profile Update username or avatar 🔒 Auth
GET /api/user/applications List your job applications 🔒 Auth
POST /api/user/applications Apply for a job 🔒 Auth
GET /api/user/saved Get saved jobs 🔒 Auth
POST /api/user/saved Save a job for later 🔒 Auth
DELETE /api/user/saved/:jobId Remove a saved job 🔒 Auth
GET /api/user/stats Application statistics 🔒 Auth
Jobs (Public)
GET /api/jobs Search and list jobs with filters 🔓 Public
GET /api/jobs/:id Get a single job by ID 🔓 Public
Scholarships (Public)
GET /api/scholarships Search scholarships 🔓 Public
GET /api/scholarships/:id Get a scholarship by ID 🔓 Public
Admin (requires admin JWT)
GET /api/admin/users List all users 👑 Admin
PATCH /api/admin/users/:id/role Change a user's role 👑 Admin
DELETE /api/admin/users/:id Delete a user permanently 👑 Admin
POST /api/admin/jobs Create one or many jobs (JSON body) 👑 Admin
GET /api/admin/jobs List all jobs (admin view) 👑 Admin
PATCH /api/admin/jobs/:id Update a job 👑 Admin
DELETE /api/admin/jobs/:id Delete a job 👑 Admin
POST /api/admin/scholarships Create one or many scholarships 👑 Admin
PATCH /api/admin/scholarships/:id Update a scholarship 👑 Admin
DELETE /api/admin/scholarships/:id Delete a scholarship 👑 Admin
Interview (AI Prep)
POST /api/interview/generate Generate AI interview prep (Groq, Gemini, Anthropic) 🔒 Auth
GET /api/interview/history Get your AI prep history 🔒 Auth
Scraper
POST /api/scraper/trigger Manually trigger the scraper (runs in background) 🔒 Auth

⚠️ This fires a long-running background process. The endpoint returns immediately with a confirmation message.

GET /health System health check 🔓 Public

Required Environment Variables

DATABASE_URL — PostgreSQL connection string
JWT_SECRET — JWT signing secret
JWT_REFRESH_SECRET — Refresh token secret
GOOGLE_CLIENT_ID — Google OAuth client ID
SMTP_HOST / SMTP_PASS — Email config
FRONTEND_URL — Frontend app URL
CRON_SECRET — Secret for cron-job.org
REDIS_URL — Redis connection URL (optional)
